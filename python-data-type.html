Python Data Types#

In programming we classify data as types for variables.
Variables can store data of different types, and different types can do different things.
Python has the following data types built-in by default, in these categories:

    Text Type:	str 
Numeric Types:	int, float, complex 
Boolean Type:	bool
Binary Types:	bytes, bytearray, memoryview

Sequence Types:	list, tuple, range 
Mapping Type:	dict
Set Types:	    set, frozenset
None Type:	    NoneType

Getting the Data Type
You can get the data type of any object by using the type() function:

Example
Print the data type of the variable e:

e = 2.718
print( type(e))

<class 'float'>

Setting the Data Type
In Python, the data type is set automatically when you assign a value to a variable:

a = 'Hello ITCS-159'; print( type(a))

<class 'str'>

a = 20; print( type(a))

<class 'int'>

a = ["apple", "banana", "cherry"]; print( type(a))

<class 'list'>

a = {"name" : "John", "age" : 36}; print( type(a))

<class 'dict'>

a = True; print( type(a))

<class 'bool'>

a = b"Hello"; print( type(a))

<class 'bytes'>

Python Number#

Three numeric types in Python

  • int

  • float

  • complex

Variables of numeric types are created automatically when you assign a value to them:

temp = 25    # int
e = 2.718  # float
z = 1+1j   # complex

To verify the type of any object in Python, use the type() function:

Example:

print( type(temp))
print( type(e))
print( type(z))

<class 'float'>
<class 'complex'>

Type Conversion#

You can convert from one type to another with the int(), float(), and complex() methods:

Example
Convert from one type to another:

x = 7    # int
y = 3.14  # float
z = 4j   # complex

#convert from int to float:
a = float(x)

#convert from float to int:
b = int(y)

#convert from int to complex:
c = complex(x)

print(a)
print(b)
print(c)

print(type(a))
print(type(b))
print(type(c))
